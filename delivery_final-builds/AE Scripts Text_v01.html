Viewport Colors

Viewport Colors is a scripts package for C4D to improve your viewport workflow. With this scripts you can apply more quickly different type of colors to your objects in viewport for reference purposes without lost your C4D materials or 3rd party render engine materials.

{{image label="Viewport Colors Features"}}

Features:

-You can use it to have a color reference of the scene to avoid wasting time in correctly viewing the materials in the viewport.

-You can add a color that you prefer to one or more selected objects (support one object, multiple objects and type of objects with they childrens too).

-You can add a solid color, Xray/Transparent color, wireframe color and random colors.

-You can create 2 type of colors variations from your key color: random variation and adjacent colors variation.

-Don't worry! You don't lose your render materials and you can use it with materials from 3rd party render engines too just like Octane Render, Arnold Renderer, Redshift, etc.

 

The package include:

-VP Custom Color scripts.

-VP Direct Color scripts.

-User guide for both scripts packages.

-7 explainer tutorials.

VP Custom Color

With VP Custom Color you can add a custom color that you prefer to one or more selected objects (support one object, multiple objects and type of objects with they childrens too).

VP Direct Color

The VP Direct Color is for a more quickly and direct workflow, you can apply one color from a preset color library to one or more selected objects (support one object, multiple objects and type of objects with they childrens too).

 

Free Trial

Also, you can download the trial version that contain only the VP Custom Color and Clear VP Color scripts.

Limitations

You can't have a children solid/transparent inside of a wireframe parent because c4d doesn't support different type of visualizations from children objects when the parent have a display tag.

Install Guide

To install the script package you need to copy the VP Color folder from “03_Scripts Folder” to c4d scripts folder:

WIN: ./C4D RXX root folder / library / scripts

OSX: ./C4D RXX root folder / library / scripts

Compatible with: R18, R17, R16 versions (not tested in older versions, maybe works fine)
